# business_site

FastAPI business website for Breakfast Technologies.

---

## Local Development

Before committing any changes, run the site locally to visually verify the changes are working as expected.

**Activate the virtual environment:**
```bash
source .venv/bin/activate
```

**Start the local server:**
```bash
uvicorn main:app --reload
```

Then open `http://localhost:8000` in your browser to review the changes before committing.
